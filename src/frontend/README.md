# AI Master

A privacy-focused AI chat application built on the Internet Computer.

## Custom Domain Configuration

The custom domain slug is configured in `frontend/deploy/custom-domain.json`.

**Current slug:** `ai-master`

### Domain Requirements
- Must be between 5-50 characters
- Can only contain letters, numbers, and hyphens
- No spaces, dots, or special characters allowed

**Important:** Do not modify the slug value without ensuring it meets the platform's domain validation requirements.
